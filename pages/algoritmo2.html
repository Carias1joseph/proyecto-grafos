Algortimo de Busqueda Binaria

Eficiente para encontrar un elemento dentro de una lista ordenada, funcionando al dividir repetidamente la lista en mitades y comparar el valor buscado con el elemento central. Ayuda a optimizar el acceso a la información en programas y bases de datos, reduciendo tiempo y recursos en búsquedas.

¿Qué es la Búsqueda Binaria?

La búsqueda binaria es un algoritmo eficiente para encontrar un elemento en una lista ordenada, reduciendo el espacio de búsqueda a la mitad en cada paso.

¿Cómo funciona?

1. Se ordena la lista (si no lo está). 2. Se compara el valor buscado con el elemento central. 3. Si es igual, se ha encontrado. 4. Si es menor, se repite la búsqueda en la mitad izquierda. 5. Si es mayor, se repite en la mitad derecha. Este proceso se repite hasta encontrar el valor o que no haya más elementos.

Características

  • Requiere lista ordenada.
  • Complejidad O(log n), mucho más rápido que la búsqueda lineal.
  • No recorre todos los elementos, solo los necesarios.

¿Para qué se usa?

Ideal para grandes volúmenes de datos ordenados, como diccionarios, bases de datos indexadas o listas estáticas.

Ventajas

  • Muy rápida en listas grandes.
  • Menos comparaciones que otros métodos.

Desventajas

  • Solo funciona con listas ordenadas.
  • Necesita cálculo de índices y división repetida.
